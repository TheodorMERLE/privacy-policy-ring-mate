# Politique de Confidentialité - RingMate

Dernière mise à jour : 13 mars 2026

Cette politique de confidentialité décrit comment l'application RingMate gère vos données personnelles.

### 1. Collecte de données
L'application ne collecte, ne stocke, ni ne transmet aucune donnée personnelle vers des serveurs externes. Toutes les données relatives à vos sessions de port sont stockées localement sur votre appareil.

### 2. Accès au stockage
L'application demande l'accès au dossier "Téléchargements" uniquement pour vous permettre d'exporter vos propres données (format fichier). Aucune autre donnée n'est lue ou modifiée.

### 3. Services tiers
Nous n'utilisons aucun outil d'analyse (Analytics), de publicité (AdMob) ou de suivi tiers.

### 4. Modifications
Nous nous réservons le droit de mettre à jour cette politique. Toute modification sera reflétée par la date de mise à jour.
